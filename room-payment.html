🔗 เชื่อมต่อระบบ Real-time

ต้องทำครั้งแรกครั้งเดียว หลังจากนั้นทุกคนในห้องจะเห็นข้อมูลเดียวกันแบบ real-time โดยไม่ต้องทำซ้ำอีก

1 สร้าง Google Sheet ใหม่
ไปที่ sheets.new → สร้าง Sheet ใหม่ → ตั้งชื่อตามใจชอบ
2 เพิ่ม Apps Script
ใน Sheet → เมนู Extensions → Apps Script
ลบโค้ดเดิมทิ้ง แล้ววางโค้ดจากไฟล์ Code.gs ที่ได้รับ

จากนั้นกด Deploy → New deployment → Type: Web app
→ Execute as: Me → Who has access: Anyone → Deploy
3 วาง URL ที่ได้
คัดลอก Web App URL ที่ได้ (ขึ้นต้นด้วย https://script.google.com/macros/s/...)
แล้ววางด้านล่างนี้
❌ เชื่อมต่อไม่ได้ ตรวจสอบ URL แล้วลองใหม่
